1. Variables
* A variable is a container (storage area) used to hold data.  
* Each variable should be given a unique name (identifier). 

```
class Main {

   public static void main(String[] args) {
      // Variables
       String name = "Aman";
       int age = 30;

       String neighbor = "Akku";
       String friend = neighbor;
   }
}

```
